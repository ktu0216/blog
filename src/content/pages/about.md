**Saegida**는 시스템을 직접 만들며 배운 것들을 기록하는 기술 블로그입니다.

인프라(서버·네트워크·홈랩), 개발 워크플로, 그리고 AI 도구를 실제로 구축하고 운영하면서 부딪힌 과정과 결론을 실전 위주로 정리합니다. "따라 하면 동작하는" 수준의 구체성을 지향합니다.

## 다루는 주제

- **인프라 / 홈랩** — TrueNAS, Tailscale, Syncthing, SSH·인증
- **개발 / 자동화** — Astro 정적 블로그, 배포 파이프라인
- **AI 도구** — Claude Code, Discord 연동, 에이전트 워크플로

## 이 블로그에 대해

[Astro](https://astro.build/)로 만들고 GitHub Pages에 배포합니다. 테마는 [AstroPaper](https://github.com/satnaing/astro-paper)를 기반으로 커스터마이즈했습니다.

글에 등장하는 경로·계정·키 등 민감 정보는 모두 더미 값으로 대체되어 있습니다.
