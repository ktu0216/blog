`draft: true`라서 `npm run build`(프로덕션)에는 포함되지 않습니다. `npm run dev`
(개발)에서는 보이므로 작성·검토 후 `draft: false`로 바꿔 발행하세요.

> 진짜 개인 메모는 블로그가 아니라 Obsidian 볼트(`code/ktu-docs`)에 두는 걸 권장합니다.
